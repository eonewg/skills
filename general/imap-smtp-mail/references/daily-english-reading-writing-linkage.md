# Daily English email: reading-writing linkage refinement

Session learning: the user asked to optimize the recurring daily English email after checking Zhihu-style考研英语阅读/写作方法. The useful durable lesson is not the specific posts, but the class-level format improvement for this recurring email.

## Direction

Keep the email as a readable daily email/handout, not a full course page. Upgrade it from an “advanced-expression magazine” into a “reading-writing linkage training card”:

- less pure expression listing;
- more 真题阅读意识: 定位句, 同义改写, 错因分类, 作者态度, 例证/转折逻辑;
- explicitly show how reading observations transfer into作文功能句.

## Durable structure refinements

Preserve the 8-section structure, but implement the sections like this:

1. 今日高级英语表达
   - 6–8 expressions, with 4–6 new + 2 review callbacks.
   - Each item needs 中文解释, Example, 中文理解, and `适用场景` such as 阅读理解 / 作文原因段 / 总结段.

2. 今日高级句型
   - 3–4 sentence patterns.
   - Add `句型功能`: 让步, 因果, 评价, 总结, 开头点题, 段末收束, etc.

3. 阅读长难句拆解
   - Keep 主干 / 修饰 / 逻辑关系 / 中文理解 / 可迁移写法.
   - Add `可能考点/命题陷阱`: 作者态度, 转折后重点, 例子支撑观点, 熟词僻义, 同义改写, etc.

4. 阅读与写作微技能
   - Replace old pure “实用写作技巧”.
   - Give 2–3 compact techniques connecting 真题阅读 and writing output.
   - Good examples: 定位句→同义改写, 例证题看观点不看例子, 态度词识别, 错因分类, 把阅读句改成作文功能句.
   - Avoid generic writing chicken soup or bulky templates.

5. 今日高级连接词
   - Continue as a separate section from replacements.
   - Group by function: 递进 / 转折 / 因果 / 结论.
   - Add `适用位置`: 原因分析, 让步转折, 段末总结, etc.
   - Hard style rule: no table, no horizontal rules, no row borders.

6. 自然高级替换
   - Continue as a separate section from connectors.
   - Format as 普通表达 → 高级表达 + 适用位置 + example.
   - Prefer items useful for reading-to-writing transfer: 同义改写, 态度评价, 原因机制, 主旨概括.
   - Hard style rule: no table, no horizontal rules, no row borders.

7. 今日可背高级段落
   - 120–180 words; reuse at least 3 expressions/patterns including at least one review callback.
   - Add 2–3 `可拆用句` and state what作文 paragraph/function each can serve.

8. 3 分钟主动回忆
   - Not only translation.
   - Use three mini tasks: one Chinese→English output prompt, one reading-logic judgment, one synonym-rewrite or ordinary→advanced sentence upgrade.
   - Provide reference answers.

## Pitfalls to avoid

- Do not revert section 04 to pure “实用写作技巧”; it should remain reading-writing linkage.
- Do not turn the daily email into a massive true-question精读课. It should stay light enough for daily reading.
- Do not add teacher/resource recommendations or platform/source names into the email body; those belong in research notes, not the lesson.
- Do not use tables or horizontal separators for sections 05/06; the user found the line-heavy layout visually noisy.
