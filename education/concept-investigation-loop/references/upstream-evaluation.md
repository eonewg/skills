# Upstream evaluation notes

Source: https://github.com/SarkAzia/baiyueguang-learning-skill

## What was absorbed

Useful core:

- Convert a knowledge point from a static definition into an investigative object.
- Ask origin, prerequisites, relationship network, dependent concepts, and next clue.
- Distinguish repeated/rephrased facts from genuinely new information.
- Use curiosity and relationship-tracing to replace passive memorization.

## What was deliberately not absorbed

- The meme-heavy “视奸前任白月光” shell as a long-term system phrase.
- Generic copy-paste prompt style.
- Treating investigation as a complete study system.

## Adaptation decision for the user

Use as a concept-understanding starter for math, 408, and CS concepts. Do not use it as a replacement for problem practice, active recall, spaced review, or exam-oriented error analysis.
