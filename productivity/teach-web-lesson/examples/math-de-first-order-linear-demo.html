本页目标识别 + 标准形
下一步伯努利方程

学完你要拿到什么

你不需要一上来背一堆方程类型。先把一阶线性这类高频基础型抓稳：它长得像 \( y' + P(x)y = Q(x) \)。

一句话抓手：未知函数 \(y\) 和导数 \(y'\) 都只出现一次，没有 \(y^2\)、\(yy'\)、\(\sin y\) 这种非线性东西。

核心概念

一阶线性微分方程的关键不是“公式很长”，而是先把题目化成标准形。标准形右边只允许是 \(x\) 的函数。

看到什么

同时有 \(y'\) 和 \(y\)，并且它们都是一次。

立刻做什么

先除掉 \(y'\) 前面的系数，整理成 \(y' + P(x)y = Q(x)\)。

公式卡

一阶线性微分方程通解

\[ y=e^{-\int P(x)\,dx}\left(\int Q(x)e^{\int P(x)\,dx}\,dx+C\right) \]
标准形式
\(y' + P(x)y = Q(x)\)
适用条件
未知函数 \(y\) 与 \(y'\) 均为一次，且系数只依赖 \(x\)。
识别关键词
\(y'\)、\(y\)、无 \(y^2\)、无 \(yy'\)、无 \(\sin y\)。
易错点
公式前必须先化标准形，不要漏除 \(y'\) 的系数。

例题拆解

判断并求解：\( y' + 2y = e^{-x} \)
第一眼判断
它已经是 \(y' + P(x)y = Q(x)\)，其中 \(P(x)=2\)，\(Q(x)=e^{-x}\)。
代入通解
\(e^{-\int 2\,dx}=e^{-2x}\)，积分因子部分为 \(e^{2x}\)。
整理答案
\[y=e^{-2x}\left(\int e^{-x}e^{2x}\,dx+C\right)=e^{-x}+Ce^{-2x}\]
展开复盘
这题没有难点，重点是确认它已经是标准形。真正容易错的是题目给成 \(a(x)y'+b(x)y=c(x)\) 时忘记先除以 \(a(x)\)。

易错点

没化标准形

看到 \(x y' + y = x^2\) 不能直接套，要先除以 \(x\)。

把伯努利认成线性

\(y' + P(x)y = Q(x)y^n\) 当 \(n\ne0,1\) 时不是一阶线性。

积分常数乱放

最后统一写一个 \(C\)，别在中间积分反复加常数。
